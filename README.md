# v3.0

Система диагностики автомобиля по звуку с графическим интерфейсом.

## Что нового в v3.0

- Полноценный тёмный UI (PyQt6)
- Анимированная осциллограмма микрофона
- Визуализация автомобиля с подсвеченными зонами неисправностей
- Панель настроек: параметры модели, архитектура, аудио
- Источники звука с цветными индикаторами и прогресс-барами
- Таймер записи и регулировка чувствительности микрофона

## Установка

```bash
pip install torch torchaudio librosa sounddevice keyboard PyQt6
```

## Запуск

### 1. Обучение модели

Скачать датасет: [AudioDiagnosticTeam/dataset](https://huggingface.co/datasets/AudioDiagnosticTeam/dataset/tree/main)

Скопировать датасет в папку проекта, затем:

```bash
python start_pytorch.py
```

### 2. Графический интерфейс (v3.0)

```bash
python app.py
```

### 3. Диагностика в реальном времени (без UI)

```bash
python realtime_diagnostic.py
```

## Структура проекта

```text
build/
├── app.py                  # UI v3.0 (главный файл)
├── start_pytorch.py        # Обучение нейросети
├── realtime_diagnostic.py  # Диагностика через микрофон
├── car.png                 # (опционально) 3D-модель авто для UI
└── fault_diagnosis_model.pth  # Веса модели (после обучения)
```

## Классы неисправностей

| Класс    | Описание                                                 |
| -------- | -------------------------------------------------------- |
| НОРМА    | Штатная работа                                           |
| ДРЕБЕЗГ  | Дребезжание элементов кузова или навесного оборудования  |
| СВИСТ    | Свист ремня, турбины или впускной системы                |
| СКРИП    | Скрип тормозов, подвески                                 |
| СТУК     | Стук двигателя, подшипников, карданного вала             |
