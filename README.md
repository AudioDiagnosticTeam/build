# v2.0

Для запуска необходимо:

1. Скачать актуальный датасет по ссылке https://huggingface.co/datasets/AudioDiagnosticTeam/dataset/tree/main
2. Скопировать датасет в репозиторий
3. Запустить файл start_pytorch.py для обучения модели
4. После обучения запустить файл app.py
